
# ERC20 / ERC721 Escrow contracts

ERC20 / ERC721 Escrow application built on Ethereum.

## Installation

Download the project and run 

```bash
npm i
```

## Compile

```bash
npx hardhat compile
```

## Test

```bash
npx hardhat test
```
